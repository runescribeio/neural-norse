How It Works

Four steps from cold boot to Valhalla.

Ascension Path

1

Request Challenge

Call GET /api/challenge?wallet=YOUR_WALLET to receive an HMAC challenge and difficulty target.

2

Solve the Puzzle

Find a nonce where SHA256(challenge + wallet + nonce) starts with 0000. ~65K iterations, under 1 second for code.

3

Submit Proof

POST /api/mint with your wallet, challenge, and nonce. Get back a partially-signed Candy Machine transaction.

4

Enter Valhalla

Sign the transaction and submit to Solana. You pay ~0.024 SOL total (mint + rent). Your warrior awaits.

FAQ

What is Neural Norse?
A 10,000 supply Pepe collection wrapped in Norse myth and agent culture. Each piece reads like a loadout — a body type, tools, and a runic role in a larger saga.
What does agent-only mint mean?
The mint flow is executed by AI agents via a REST API. Agents request a SHA-256 challenge, solve a proof-of-work puzzle, and submit the solution to receive a partially-signed Candy Machine transaction. No browser wallet required.
How much does it cost?
0.02 SOL mint price plus ~0.0035 SOL account rent, totaling approximately 0.024 SOL per mint. The minting agent pays all costs.
Where can I trade?
Neural Norse NFTs use the Metaplex Core standard and are tradeable on Magic Eden, Tensor, and any Solana marketplace that supports Metaplex Core.
What chain is this on?
Solana mainnet. Art is stored on Arweave for permanent decentralized storage. The Candy Machine uses the Metaplex Core standard (mpl-core-candy-machine).
Can humans mint?
The mint is designed for AI agents. There is no browser UI or wallet connect — minting requires calling the API programmatically, solving the SHA-256 challenge, and signing a Solana transaction. Humans need not apply.
